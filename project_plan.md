# Puneeth Kumar — Personal Portfolio Website

## 1. Project Description
A professional single-page portfolio website for Puneeth Kumar. The site showcases his background, skills, work process, and provides a direct contact channel for potential clients or employers.

**Target Users:** Recruiters, hiring managers, potential clients, collaborators
**Core Value:** Present Puneeth Kumar as a credible, skilled professional and invite contact

---

## 2. Page Structure
- `/` — Home (single-page with anchor sections)
  - `#home` — Hero / Landing
  - `#purpose` — About / Purpose
  - `#key-features` — Key Features / Skills
  - `#how-it-works` — How It Works / Process
  - `#benefits` — Benefits
  - `#contact` — Contact Form
